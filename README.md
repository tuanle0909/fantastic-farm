# Fantastic Farm — Web Client

Vite + React + Sui wallet (`@mysten/dapp-kit`). Deploy as a **static site** (`npm run build` → `dist/`).

- **Deploy:** see [`DEPLOY.md`](./DEPLOY.md) (Vercel, Netlify, env vars, GitHub Packages).
- **API:** set `VITE_BE_API_URL` to your backend (e.g. Render).

Shared constants package: `@fantastic-farm/shared` → `@tuanle0909/fantastic-farm-shared` on GitHub Packages (see committed [`.npmrc`](./.npmrc)).
